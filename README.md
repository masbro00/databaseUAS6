# Sistem Informasi Manajemen Kost (SIMKOST)

Repository ini merupakan proyek UAS Mata Kuliah **Pemrograman Basis Data**.

## Deskripsi
SIMKOST adalah sistem basis data yang digunakan untuk mengelola informasi rumah kost, mulai dari pemilik, data kost, kamar, penyewa, pembayaran, hingga fasilitas kamar.

Database dibangun menggunakan **MySQL/MariaDB** dan telah memenuhi materi yang dipelajari selama satu semester.

## Fitur Database
- Struktur database relasional
- 9 tabel utama
- Relasi One-to-One
- Relasi One-to-Many
- Relasi Many-to-Many
- Data seeder
- Function
- Stored Procedure
- Trigger
- View
- User Management & Privilege
- Backup database (.sql)

## Daftar Tabel
- owner
- kost
- room
- facility
- room_facility
- tenant
- tenant_profile
- rental
- payment

## Tools
- XAMPP
- phpMyAdmin
- MySQL / MariaDB

## Cara Menggunakan
1. Buka phpMyAdmin.
2. Buat database baru (opsional jika file SQL sudah membuat database).
3. Import file `db_manajemen_kost.sql`.
4. Database siap digunakan.
